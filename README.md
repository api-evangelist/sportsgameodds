# SportsGameOdds

SportsGameOdds API provides real-time sports betting odds data from 80+ sportsbooks across 50+ leagues. The API delivers live and pre-match odds across 3,000+ markets including moneylines, spreads, over/unders, team props, and player props for NFL, NBA, MLB, NHL, NCAAF, NCAAB, EPL, UCL, UFC, PGA, ATP and more. Features include consensus odds calculations, streaming for live data, an interactive data explorer, and official SDKs for JavaScript/TypeScript, Python, Java, Go, and Ruby.
